◆ getValidSubResources()

static Infinity\Api\Resources\ResourceAbstract::getValidSubResources ( )
static

This returns the valid relations of this resource. Definition of what is allowed to chain after this resource. Make sure to add in this method when adding new sub resources. Example: $client->ticket()->comments(); Where ticket would have a comments as a valid sub resource. The array would look like: ['comments' => '\Infinity\Api\Resources\TicketComments']
